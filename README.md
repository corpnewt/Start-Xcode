# Start-Xcode
Bash script to directly launch Xcode via its binary

```
usage: Start-Xcode.command [-h] [-p] [-x PATH]

Start-Xcode - a bash script to directly launch Xcode via its binary

optional arguments:
  -h, --help              show this help message and exit
  -p, --prompt            override xcode-select detection and prmopt for the
                          Xcode.app path
  -x PATH, --xcode PATH   provide an explicit path to the target Xcode.app
                          - overrides -p
```
